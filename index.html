When I just get introduced to the multi-agent system (MAS), in 2017, unsurprisingly there isn’t much introductory articles, blogs, or YouTube videos. MAS were still very much under-explored in literature. After all, we are still working on perfecting intelligent agent. We can surely thinking about connecting them later…… (I guess I’m not the only one who procrastinate) Well, unsurprisingly, lately OpenAI(need href) and Deepmind (need href) both investigate on doing research on MAS. Perhaps we are not very far away from coordinating a group of agents?

Jacques Ferber defined that an agent can be a physical or virtual entity that can act, perceive its environment (in a partial way) and communicate with others, is autonomous and has skills to achieve its goals and tendencies. [Ferber, J. (1999). Multi-agent systems: an introduction to distributed artificial intelligence (Vol. 1). Reading: Addison-Wesley.] As you can tell, this is an extremely loose definition. This is because an agent can be instantiate in a lot of situations. // A common use would be digital society where each agent is considered an individual who are of some characteristics and interact with other individual as how they are defined. These agents can also work as a proxy of a party and negotiate with other agent in order to make a deal. The list goes on a on. // A multi-agent system put multiple agents in a predefined environment. The environment can be dynamic or static depends on the usage. The agent can also be connected in a specific topology.
